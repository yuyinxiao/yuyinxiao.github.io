算法导论

动态规划

  • 最优子结构性质。如果问题的最优解所包含的子问题的解也是最优的。
  • 无后效性。即子问题的解一旦确定，不受之后和包含它的更大的问题的决策影响。
  • 子问题重叠性质。子问题会被重复计算多次。动态规划对每一个子问题只计算一次保存在表格，简化重复计算时。
